CSS CURSOR

CSS cursor property used to define cursor type (i,e. mouse pointer) when the mouse moves over a certain area or, over a link on the webpage

EXAMPLE..!

THIS IS FIRST PARAGRAPH..! (AUTO)

THIS IS FIRST PARAGRAPH..! (DEFAULT)

THIS IS FIRST PARAGRAPH..! (NONE)

THIS IS FIRST PARAGRAPH..! (CONTEXT_MENU)

THIS IS FIRST PARAGRAPH..! (HELP)

THIS IS FIRST PARAGRAPH..! (POINTER)

THIS IS FIRST PARAGRAPH..! (PROGRESS)
